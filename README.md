# qa.automationexercise-web.webdriverio


### Pré-Requisitos do Projeto

Antes de executar os testes, certifique-se de que os seguintes softwares estão instalados no seu sistema:

[Node JS](https://nodejs.org/pt/download)

[WebdriverIO](https://webdriver.io/docs/gettingstarted/)



## Executando os Testes:
Para executar os testes, siga estes passos:

1. Clone o repositório:

```powershell
git clone https://github.com/Calazhen/qa.automationexercise-web.webdriverio.git
```

2. Navegue para o diretório do projeto:

```powershell
cd qa.automationexercise-web.webdriverio
```

3. Execute os testes:

```powershell
npm run wdio --feature
```
4. Gere o relatório do Allure:

```powershell
npx allure serve allure-results --clean -o allure-report
```
